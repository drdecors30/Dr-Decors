Steps to Reset the Admin Email and Password

Step 1: Open the Command Prompt
	Open Command Prompt (or Terminal) in the root directory of your application.

Step 2: Generate a New Password Hash
	Run the following command, replacing `YOUR_NEW_PASSWORD` with the password you want to use:

node -e "const crypto=require('crypto'); console.log(crypto.createHash('sha256').update('YOUR_NEW_PASSWORD'+'deedecors-salt-key-9283').digest('hex'));"


Step 3: Copy the Generated Hash
	After executing the command, a SHA-256 hash will be displayed. Copy the generated hash.

Step 4: Update the Admin Credentials
	Replace the existing admin email and password hash in the following files:
	> data/db.json
	> src/server/db.ts

> Update the admin email with the new email address.
> Replace the existing password hash with the newly generated hash.
> Save the files and restart the application for the changes to take effect.
